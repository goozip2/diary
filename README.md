## Project Outline:

### Project Name:

> 일기를 통한 멘탈 케어 서비스

### Project Goal:

> 일기에 적혀져 있는 문장을 분석하여 사용자의 감정을 파악하고 그에 맞는 맞춤 컨텐츠를 제공

### Project Date:

> 2023.01.30 ~ 2023.02.28

### Implement Goal:

> 1. 작성한 일기를 분석하여 감정과 관련 상황을 알아내 그에 맞는 플레이리스트와 관련 키워드 사이트를 제공한다..
> 2. 분석한 일기 데이터를 모아 통계자료를 제공하여 사용자의 최근 한 달의 감정상태와 관심사를 알려준다.
